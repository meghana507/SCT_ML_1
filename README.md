# House Price Prediction using Linear Regression

## Project Overview
This project implements a **Linear Regression** model to predict housing prices. By analyzing key features of a property, the model estimates the market value, providing insights into real estate pricing trends.

## Dataset
The model is trained on the **House Price Dataset**, focusing on the following features:
* **Square Footage:** The total liveable area of the house.
* **Bedrooms:** The number of bedrooms available.
* **Bathrooms:** The number of bathrooms available.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn

## Implementation Steps
1. **Data Preprocessing:** Handling missing values and feature scaling.
2. **Exploratory Data Analysis (EDA):** Visualizing correlations between square footage and price.
3. **Model Training:** Implementing `LinearRegression` from Scikit-learn.
4. **Evaluation:** Assessing performance using Mean Squared Error (MSE) and R-squared metrics.
